1. Create git server first  
 git init 
  
 2. Configure the git account, to check whether username exist or not in git config list 
 git config --list 
  
 3. Configure the username 
 git config --global user.name 'username' 
  
 4. configure the email id 
 git config --global user.email 'email id' 
  
 5. establish connection between cloude repo and local server 
 git remote add origin urlOfRemote 
  
 6. to remove the remote connection 
 git remote remove origin 
  
 7. to check the remote connection 
 git remote -v 
  
 8. to add the file into staging area 
 git add fileName (To add single or multiple file) 
 git add .  (To add all the file into staging area) 
  
 9. to remove the file from staging area 
 git rm --cached fileName (to remove single or multiple file from staging area) 
 git reset HEAD -- . (to remove the all the file from staging area) 
  
 10. to check the status of the staging are 
 git status 
  
 11. to bring from staging area to commit area 
 git commit -m 'Comment' 
  
 12 to change the branch from master to main 
 git branch -m master main 
  
 13. to push to the remote repository as earlier named origin and main brach 
 git push -f origin main 
  
 14. to set the upstream 
 git branch --set-upstream-to=origin/main
